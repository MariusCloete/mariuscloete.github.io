K3s install on Rock64

This is a guide on how to install K3s on a Rock64. K3s is a lightweight Kubernetes distribution that is easy to install and run. It is a great way to get started with Kubernetes.

To start you can download the installscript from here (K3s.io)

curl -sfL https://get.k3s.io | K3S_KUBECONFIG_MODE="644" sh -

This will install K3s and create a kubeconfig file in /etc/rancher/k3s/k3s.yaml. You can use this file to access the Kubernetes cluster.
To test your install run the following command:

kubectl get nodes

This should return the node you installed K3s on.

Next we need to configure the iptables to allow traffic to the pods and reboot. Run the following command:

iptables -F; update-alternatives --set iptables /usr/sbin/iptables-legacy; update-alternatives --set ip6tables /usr/sbin/ip6tables-legacy; reboot

After the reboot get the node token for master:

cat /var/lib/rancher/k3s/server/node-token

Now you can join the worker nodes to the master using the following command:

curl -sfL https://get.k3s.io | K3S_URL=https://[master-ip]:6443 K3S_TOKEN=[node token] K3S_NODE_NAME=[nodename] sh -

You can now run the following command on your master node to check the nodes:

kubectl get nodes

This should return all the nodes in your cluster.
