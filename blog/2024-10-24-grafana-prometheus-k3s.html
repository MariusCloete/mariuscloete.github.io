Install instruction for Grafana and Prometheus on K3s

This is a guide on how to install Grafana and Prometheus on K3s.

Add the helm charts to you repo:
helm repo add prometheus-community https://prometheus-community.github.io/helm-charts

Install prometheus-stack:
helm install kube-prometheus-stack --create-namespace --namespace prometheus prometheus-community/kube-prometheus-stack

Confirm everything is running:
kubectl get pods -n prometheus

Make a backup of the Grafana service that was created by defualt and then delete it:
kubectl get svc kube-prometheus-stack-grafana -n prometheus -o yaml > kube-prometheus-stack-grafana.yaml

kubectl delete -f kube-prometheus-stack-grafana.yaml

Create and expose a new Grafana service:
kubectl expose deployment kube-prometheus-stack-grafana --type=LoadBalancer -n prometheus

Confirm the Grafana service is running:
kubectl get svc kube-prometheus-stack-grafana -n prometheus

Get the external IP and port of the Grafana service:
kubectl get svc kube-prometheus-stack-grafana -n prometheus -o jsonpath='{.status.loadBalancer.ingress[0].ip}'; echo

kubectl get svc kube-prometheus-stack-grafana -n prometheus -o jsonpath='{.spec.ports[0].port}'; echo

Once you have the IP and port you can access Grafana in your browser:
User: admin
Password: prom-operator
