Helm with k3s

So you would like to use Helm with k3s?
Then the first thing you need to do is install Helm.
After that you need to link Helm to k3s.

To Link Helm to k3s you need to run one command:
KUBECONFIG=/etc/rancher/k3s/k3s.yaml

Do quick test:
helm ls --all-namespaces
